Как запустить

1

Скачайте бинарь

Возьмите файл для своей системы выше. Один файл, без установщика.

2

Установите ffmpeg

Нужен для перекодирования некоторых форматов на лету.

# Windows
winget install ffmpeg

# macOS
brew install ffmpeg

# Linux (Debian/Ubuntu)
sudo apt install ffmpeg
3

Запустите бинарь

Двойной клик — или из терминала. Браузер откроется автоматически на http://localhost:7777. Дальше — поиск, постеры, плеер.

Из браузера на телевизоре

Если телевизор и компьютер в одной WiFi-сети — можно смотреть прямо с TV. Запустите ZubraCinema на ноуте или мини-ПК, узнайте его локальный IP (например, 192.168.1.42), и откройте на телевизоре в браузере адрес http://192.168.1.42:7777.

ПК становится домашним стриминговым сервером. Управление — пультом телевизора через его встроенный браузер, либо с телефона на ту же страницу. Никаких аккаунтов, никаких облаков.
